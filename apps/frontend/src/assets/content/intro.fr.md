# Qu'est-ce que Docroot ?

**Docroot** est un serveur de documentation léger auto-hébergé.
Téléversez des archives ZIP versionnées et parcourez-les via
cette interface ou l'API REST.  Naviguez espace de noms →
projet → version pour ouvrir un document.

Pour la documentation complète, consultez
[docroot.readthedocs.io](https://docroot.readthedocs.io/).
