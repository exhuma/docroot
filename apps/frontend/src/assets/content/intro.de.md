# Was ist Docroot?

**Docroot** ist ein leichtgewichtiger, selbst gehosteter
Dokumentationsserver.  Laden Sie versionierte ZIP-Archive hoch
und durchsuchen Sie diese über die Web-Oberfläche oder die
REST-API.  Navigieren Sie Namensraum → Projekt → Version, um
eine Dokumentation zu öffnen.

Die vollständige Dokumentation finden Sie auf
[docroot.readthedocs.io](https://docroot.readthedocs.io/).
